# 填格子 (tile.c)

## 题目描述

有一个 $ 2\times n$ 的格子，你要用 $ 2\times 1$ 或 $ 1\times 2$ 的方块把它填满，不能有重叠区域，求所有填法的总数

## 输入格式

输入一个正整数 $n$，其中 $ 1\leqslant n\leqslant 40$

## 输出格式

输出一个整数，表示所有填法总数

## 脚注

$n=3$ 时，有这三种

```
+---+---+---+
|       |   |
+---+---+   +
|       |   |
+---+---+---+

+---+---+---+
|   |       |
+   +---+---+
|   |       |
+---+---+---+

+---+---+---+
|   |   |   |
+   +   +   +
|   |   |   |
+---+---+---+
## 样例
### 样例输入
3
### 样例输出
3
