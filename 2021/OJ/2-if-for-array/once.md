# 只出现了一次的数 (once.c)

## 题目描述

$n$ 个整数中有 $n-1$ 个整数出现了两次，有 $ 1$ 个整数只出现了一次，找出那个只出现了一次的整数。

## 输入格式

输入包含两行，第一行为一个正整数 $n$，$1\leqslant n\leqslant 10^5$

第二行包含 $ 2n-1$ 个正整数 $a_i$，$1\leqslant a_i\leqslant n$

## 输出格式

输出一个正整数，表示这个只出现了一次的整数

## 脚注

**你可以不用数组实现吗？**

## 样例

### 样例输入

3
1 1 2 2 3

### 样例输出

3
